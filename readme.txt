Ranked By Review latest reviews widget

== Description ==
Ranked By Review latest reviews widget displays your latest business reviews from http://rankedbyreview.co.nz on your wordpress site, for more information visit: http://amplifiernz.com

== Installation ==
Upload and install plugin
use the following shortcode:
[ranked-review-widget heading=\"Latest Reviews\" businessId=\"110307\"]
update the heading and businessId to your values.
